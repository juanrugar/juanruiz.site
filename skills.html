GIS, Web Development, WebGIS, and Data Analysis

I am a GIS analyst and web GIS developer, a profile which blends web development with the geospatial capabilities of the GIS.

Web GIS development is not far from pure web dev, but gets you acquainted with the specific libraries and APIs such as Google Maps, Leaflet.js, MaxBoxGL.js or KeplerGL, and SDI or Spatial Data Infrastructure containing free spatial data on any subject you can imagine. As in Web Dev, I can create web sites and apps with geospatial capabilities. From there I can also create simple mobile Android apps with Cordova. Some samples of webGIS projects.

As GIS analyst I use the main GIS packages (ArcGIS, QGIS, Geomedia) to provide data a spatial framework where to perform the necessary processing so as to produce a completely new piece of information. All sort of questions related to urbanism and land management are to be approached in this arena taking leverage of free data and spatial information such as satellite imagery and cartography. As part of GIS comes Python and R. Both are flagships languages of data analysis, and in the case of Python, it provides a series of great libraries to use within and along the main GIS packages. On the other hand, R as a specifically designed for analysis language allows to perform complex and thorougful data wrangling and visualization on huge data sets.
